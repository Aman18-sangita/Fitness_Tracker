#  AI Fitness Tracker

A modern **AI-powered Full-Stack Fitness Tracker** built with **React.js, Strapi, Tailwind CSS, and Google Gemini AI**. This application helps users achieve their fitness goals by tracking workouts, monitoring calorie intake and expenditure, and analyzing food images using AI to estimate calorie consumption.

## Features

- Set and manage daily fitness goals
- Track calories consumed through meals
- Monitor calories burned during workouts
- AI-powered food image analysis using Google Gemini AI
- Upload food images to estimate calorie intake automatically
- Secure user authentication (Sign Up / Sign In)
- Update and manage user profile
- Track overall fitness progress
- Responsive and modern user interface
- Full-stack deployment support
